Citron

Jedná se o projekt do předmětu VIA("Vývoj internetových aplikací"). Uživatel si pomocí mapy navolí odkud, kam a kdy chce letěte nebo doplní kdy se chce vrátit. Při stisknu na tlačítko vyhledej, stránka vrátí všechny letenky s touto volbou. Nechal jsem se dost inspirovat ze stránky KIWI.com
Použité API:

  • Skypicker Api
  • Google Maps Api
